# Изменения UI магазина

## Обзор изменений

Переработан интерфейс магазина для улучшения UX при просмотре подарков и их атрибутов.

## Что изменено

### 1. Категории магазина (Shop.jsx)
**До:** 4 кнопки - Подарок, Модель, Фон, Символ  
**После:** 2 кнопки - Подарок, Фон

- ❌ Удалена кнопка "Модель" 
- ❌ Удалена кнопка "Символ"
- ✅ Обе оставшиеся кнопки открывают фильтр при нажатии

### 2. Карточка подарка - Модели (GiftDetailsModal.jsx)

**До:** Строка с названием модели и редкостью  
**После:** Expandable полоса на всю ширину со стрелочкой

#### Особенности:
- 📋 Показывает текущую модель подарка
- ▶️ По клику разворачивается список ВСЕХ доступных моделей для этого подарка
- 📡 Модели загружаются из `https://shelloch.xyz/gifts/models_list.json`
- ✅ Текущая модель подсвечивается голубым цветом
- 🎨 Без анимаций - только названия моделей

**Структура данных models_list.json:**
```json
{
  "Spiced Wine": {
    "models": [
      "Absinthe",
      "Aqua Vita",
      "Aquarium",
      "Aries Rum",
      ...
    ]
  }
}
```

### 3. Карточка подарка - Фон (GiftDetailsModal.jsx)

**До:** Строка с названием фона и редкостью  
**После:** Кнопка со стрелочкой, открывающая модальное окно

#### Модальное окно фонов:
- 🖼️ Сетка из круглых градиентов (2-3 колонки на мобильных)
- 🎨 Каждый фон показывает:
  - Круглый градиент с реальными цветами
  - Название фона
  - Редкость (%)
- ✅ Текущий фон подсвечивается рамкой
- 📡 Фоны загружаются из `https://shelloch.xyz/gifts/backdrops.json`
- ✕ Кнопка закрытия в хедере

### 4. Карточка подарка - Символ

**До:** Строка с названием символа и редкостью  
**После:** ❌ Полностью убрана


### Новые CSS классы

**Expandable модели:**
- `.attribute-expandable` - контейнер
- `.attribute-header` - кликабельный заголовок
- `.attribute-arrow` - стрелочка (поворот при раскрытии)
- `.attribute-list` - список моделей
- `.attribute-item` - элемент модели
- `.attribute-item.current` - текущая модель (подсветка)

**Модальное окно фонов:**
- `.backdrop-modal-overlay` - затемненный фон
- `.backdrop-modal` - само окно
- `.backdrop-modal-grid` - сетка фонов
- `.backdrop-modal-item` - элемент фона
- `.backdrop-circle-large` - круглый градиент (80x80px)

## Преимущества

✅ **Упрощенная навигация** - только 2 основные категории вместо 4  
✅ **Больше информации** - пользователь видит ВСЕ доступные модели для подарка  
✅ **Визуальный выбор фонов** - градиенты вместо текста  
✅ **Меньше беспорядка** - убран неиспользуемый символ  
✅ **Лучший UX** - expandable элементы не перегружают интерфейс
