## 🚀 Deployment

### Процесс
- [ ] Git pull на сервере
- [ ] Зависимости установлены (pip install -r requirements.txt)
- [ ] Виртуальное окружение активировано
- [ ] Сервис перезапущен (systemctl restart shelloch-backend)
- [ ] Проверка статуса (systemctl status shelloch-backend)
- [ ] Логи проверены на ошибки

### Systemd Service
```ini
[Unit]
Description=Shell Gift Bot Backend
After=network.target

[Service]
Type=simple
User=www-data
WorkingDirectory=/opt/shell
Environment="PATH=/opt/shell/venv/bin"
ExecStart=/opt/shell/venv/bin/uvicorn app.run:app --host 0.0.0.0 --port 8000
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```
