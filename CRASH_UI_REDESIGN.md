# Редизайн UI краш игры

## Обзор

Полностью переработан интерфейс краш игры под минималистичный дизайн со скриншота.

## До и После

### До
- Маленький множитель (72px) с низкой opacity
- Ракета с анимацией по центру
- Линия траектории
- История раундов
- Список всех ставок игроков
- Кнопки с градиентом
- Отдельная модалка для ставок

### После
- ✅ Огромная цифра множителя (120px) по центру
- ✅ Минималистичный черный фон
- ✅ Кнопки быстрых множителей (Ожидание, x1.01, x10.00)
- ✅ Кнопки со стилями spin-button-fixed
- ✅ Панель "Ставок еще нет"
- ✅ Строка "Онлайн" и "Об игре"
- ✅ Кнопки на 5px от safe-area-inset-bottom

## Технические детали

### Цвета множителя

```javascript
const getMultiplierColor = () => {
  if (crashed) return '#ff4444'
  if (multiplier >= 10) return '#ffd700'  // Золотой
  if (multiplier >= 5) return '#ff8c00'   // Оранжевый
  if (multiplier >= 2) return '#00ff88'   // Зеленый
  return '#ffffff'                         // Белый
}
```

### Safe area для кнопок

```javascript
const safeAreaBottom = tg?.safeAreaInset?.bottom || 
                       tg?.contentSafeAreaInset?.bottom || 0

<div className="crash-controls" style={{ 
  bottom: `calc(5px + ${safeAreaBottom}px)` 
}}>
```

### Импорт стилей spin button

```css
@import './Spin.css';
```
